# Каталог подсистем VARSYS

Сюда добавляются модули следующих этапов, каждый в своей папке:

- `RadioModule/` — CC1101 sub-GHz
- `InputModule/`  — энкодер и кнопка
- `IrModule/`     — инфракрасный приёмопередатчик

Каждый модуль реализует интерфейс `IModule` (см. `src/core/Module.h`)
и регистрируется в `Core::registerModules()`.
