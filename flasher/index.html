1 · Подключение · Connect

  • Подключите устройство к ПК USB‑кабелем для данных (не «только зарядка»). · Use a data USB cable.
  • Откройте эту страницу в Chrome или Edge на компьютере. · Open in desktop Chrome/Edge.

2 · Драйверы · Drivers

T‑Embed CC1101 обычно использует нативный USB ESP32‑S3 — драйвер не нужен. Если порт не появляется, поставьте драйвер моста (CP210x / CH340·CH9102 / FTDI):

ChipWindowsmacOSLinux
CP210x CP210x VCP ↗ CP210x VCP ↗ в ядре (cp210x)
CH340 / CH9102 CH343SER ↗ · CH341SER ↗ CH34x macOS ↗ в ядре (ch341) · repo ↗
FTDI FTDI VCP ↗ FTDI VCP ↗ в ядре (ftdi_sio)

Браузер не ставит драйверы сам — скачайте, установите и переподключите кабель. · The browser can’t install drivers — install, then re‑plug.

3 · Режим прошивки · Flashing mode

Чаще всего автоматически: просто нажмите Install. Если не подключается — переведите вручную в режим загрузки:

Схема условна — расположение кнопок зависит от ревизии платы. · Schematic — exact button placement may vary by board revision.

  1. Зажмите и держите BOOT (кнопка GPIO0). · Hold BOOT.
  2. Коротко нажмите RESET (EN) и отпустите. · Tap RESET.
  3. Отпустите BOOT. · Release BOOT.
  4. Нажмите Install и выберите порт. · Click Install, pick the port.
  5. После прошивки нажмите RESET для запуска. · Press RESET to run.
