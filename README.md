# VARSYS

Модульная прошивка для **LILYGO T-Embed CC1101 Plus S3** (ESP32-S3).

## Этап 1 — Ядро + визуальный интерфейс

Текущая версия закладывает фундамент: системное ядро и графический UI.
Радио (CC1101), ИК, BLE, файлы и настройки — следующие этапы.

## Аппаратная платформа

| Компонент | Описание |
|-----------|----------|
| MCU       | ESP32-S3 (240 МГц, PSRAM) |
| Дисплей   | ST7789 170×320, SPI |
| Ввод      | Энкодер вращения + кнопка |
| Радио     | CC1101 sub-GHz (этап 2+) |

> ⚠️ Распиновка в [`src/hal/board_pins.h`](src/hal/board_pins.h) приведена по
> официальной схеме LilyGO. **Сверьте с ревизией своей платы** перед прошивкой.

## Технологический стек

- **Framework:** Arduino-ESP32 (через PlatformIO)
- **Графика:** LovyanGFX (драйвер ST7789)
- **GUI:** LVGL 8.x

## Архитектура

```
main.cpp
   └── Core (ядро, синглтон)
         ├── Logger        — уровневое логирование
         ├── EventBus      — шина событий (pub/sub)
         └── ModuleManager — реестр и планировщик модулей (IModule)
               └── UIManager (модуль визуального интерфейса)
                     ├── Display (HAL, LovyanGFX)
                     └── Screen'ы (LVGL)
                           ├── BootScreen
                           └── HomeScreen
```

Каждая подсистема реализует интерфейс [`IModule`](src/core/Module.h) и
регистрируется в ядре. Модули не зависят друг от друга напрямую — общаются
через [`EventBus`](src/core/EventBus.h). Это и есть основа модульности.

Подробнее: [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).

## Структура проекта

```
VARSYS/
├── platformio.ini        # конфигурация сборки и зависимостей
├── partitions.csv        # таблица разделов (16 МБ flash)
├── include/
│   ├── varsys_config.h   # глобальные параметры прошивки
│   └── lv_conf.h         # конфигурация LVGL
├── src/
│   ├── main.cpp
│   ├── core/             # ядро: Core, Logger, EventBus, Module, ModuleManager
│   ├── hal/              # абстракция железа: пины, дисплей
│   ├── ui/               # визуальный интерфейс: UIManager, Screen + экраны
│   └── modules/          # будущие подсистемы (радио, ИК, ...)
└── docs/
```

## Сборка и прошивка

```bash
# собрать
pio run

# прошить + открыть монитор
pio run -t upload -t monitor
```
